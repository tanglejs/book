Creating a new app

The tangle-app core plugin includes a generator to help you get started.

Demonstration

Basic usage

Interactive

$ cd myproject
$ tangle app new
# ... interactive configuration

Non-interactive

$ cd myproject
$ tangle app new --config=$(cat appConfig.json)
# ... interactive configuration

From a script

generator = require 'tangle-app/new'

generator(process.cwd).run
  name: 'blog'
  version: '0.0.0'
.progress (event) ->
  console.log event
.then (result) ->
  console.log 'App generated'
.fail (err) ->
  throw new Error 'An error occurred'

Need help?

Try $ man tangle-app-new and $ tangle help app new
